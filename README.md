**3. 🍵 构造与析构的圆舞曲**
```cpp
class TimeKeeper {
public:
    // 温柔的调酒师（默认构造）
    TimeKeeper() = default;
    
    // 定制时光的匠人（带参构造）
    TimeKeeper(int h, int m) : hour(h), minute(m) {}
    
    // 静默的清道夫（析构函数）
    ~TimeKeeper() { cout << "时辰到了哦..." << endl; }
};
```
**4. ✨ 复制艺术的二重奏**
```cpp
// 深拷贝的仪式（拷贝构造）
TimeKeeper::TimeKeeper(const TimeKeeper& tk) {
    hour = tk.hour + 1;  // 示例性的深度加工
}

// 赋值协奏曲（运算符重载）
TimeKeeper& operator=(const TimeKeeper& tk) {
    if(this != &tk) {
        hour = tk.hour;  // 要温柔地传递值呢
    }
    return *this;
}
```

**5. 📜 Const 守护者的誓约**
```cpp
int getHour() const { 
    // 这个函数不会偷偷修改任何状态哦
    return hour * 60;  // 返回分钟数更贴心呢
}
```
### 🌸 重要事项提醒
| 概念                | 注意事项                      |
|---------------------|-----------------------------|
| 默认拷贝构造        | 浅拷贝可能引发内存事故        |
| 赋值运算符          | 记得自赋值检查                |
| const成员函数       | 守护对象纯洁性的骑士          |
